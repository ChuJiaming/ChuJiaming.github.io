## 基本命令

### 文件和目录操作

```shell
pwd     #显示当前目录
ls      #显示当前目录下的文件
cd      #切换目录
mkdir   #创建目录
touch   #创建文件
rm      #删除文件
rmdir   #删除目录
cp      #复制文件
mv      #移动文件
```

### 文件内容操作

```shell
cat     #查看文件内容
```

### 文件权限操作

```shell
chmod   #修改文件权限
chown   #修改文件所有者
chgrp   #修改文件所属组
```
